# سیستم آپلود فایل با لینک دانلود

یک برنامه وب ساده برای آپلود فایل‌ها و دریافت لینک دانلود با محافظت پسورد.

## ویژگی‌ها

- ✅ آپلود فایل با پسورد
- ✅ تولید لینک دانلود یکتا برای هر فایل
- ✅ دانلود فایل با لینک
- ✅ مشاهده لیست فایل‌های آپلود شده
- ✅ محافظت با پسورد برای آپلود
- ✅ رابط کاربری فارسی و زیبا

## نصب و راه‌اندازی

### 1. نصب وابستگی‌ها

```bash
pip install -r requirements.txt
```

### 2. اجرای برنامه

```bash
python app.py
```

برنامه روی آدرس `http://localhost:5000` اجرا می‌شود.

### 3. تنظیم پسورد

1. به آدرس `http://localhost:5000/setup` بروید
2. پسورد خود را تنظیم کنید
3. حالا می‌توانید فایل آپلود کنید

## استفاده

### آپلود فایل

1. به صفحه "آپلود فایل جدید" بروید
2. فایل خود را انتخاب کنید
3. پسورد را وارد کنید
4. روی دکمه "آپلود فایل" کلیک کنید
5. لینک دانلود را دریافت کنید

### دانلود فایل

لینک دانلود به صورت خودکار بعد از آپلود نمایش داده می‌شود. می‌توانید این لینک را با دیگران به اشتراک بگذارید.

### مشاهده فایل‌ها

از صفحه "مشاهده فایل‌های آپلود شده" می‌توانید لیست تمام فایل‌های آپلود شده را ببینید.

## ساختار فایل‌ها

```
upload/
├── app.py                 # فایل اصلی برنامه
├── requirements.txt       # وابستگی‌های Python
├── password.txt          # فایل ذخیره پسورد (خودکار ایجاد می‌شود)
├── uploads/              # پوشه فایل‌های آپلود شده
├── data/                 # پوشه اطلاعات فایل‌ها
│   └── files.json        # دیتابیس فایل‌ها
└── templates/            # قالب‌های HTML
    ├── index.html
    ├── upload.html
    ├── files.html
    ├── setup.html
    ├── change_password.html
    └── error.html
```

## تنظیمات

در فایل `app.py` می‌توانید تنظیمات زیر را تغییر دهید:

- `UPLOAD_FOLDER`: پوشه ذخیره فایل‌ها (پیش‌فرض: `uploads`)
- `MAX_FILE_SIZE`: حداکثر حجم فایل (پیش‌فرض: 100 مگابایت)
- `ALLOWED_EXTENSIONS`: پسوندهای مجاز فایل
- `port`: پورت سرور (پیش‌فرض: 5000)

## امنیت

- پسورد در فایل `password.txt` ذخیره می‌شود
- برای استفاده در محیط تولید، بهتر است از WSGI server مانند Gunicorn استفاده کنید
- برای امنیت بیشتر، می‌توانید از HTTPS استفاده کنید

## اجرا روی سرور

برای اجرا روی سرور، می‌توانید از Gunicorn استفاده کنید:

```bash
pip install gunicorn
gunicorn -w 4 -b 0.0.0.0:5000 app:app
```

یا با Nginx به عنوان reverse proxy:

```nginx
server {
    listen 80;
    server_name your-domain.com;

    location / {
        proxy_pass http://127.0.0.1:5000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }
}
```
